# 🎮 Steam 게임 가격 분석

이 프로젝트는 Steam 게임 데이터를 바탕으로 게임 가격 분포를 분석하고, 가격대별 게임 수를 시각화합니다.

## 📊 분석 목표

- Steam에 등록된 게임들을 가격대별로 구분하여 **어떤 가격대에 게임이 많이 분포되어 있는지 시각화**합니다.
- 게임 시장에서 **저가 게임과 고가 게임의 분포 차이**를 파악할 수 있습니다.

## 🗂️ 데이터

- CSV 파일: `steam.csv`
- 주요 컬럼: `price` (게임 가격)
- 출처: Steam 게임 데이터셋 (자체 수집 또는 공개 데이터 활용)

## ⚙️ 사용된 도구

- Python 3.x
- pandas
- matplotlib

## 🧹 전처리 과정

1. **가격 필드 추출**  
   데이터프레임에서 `price` 컬럼을 기준으로 분석합니다.

2. **가격 구간(bin) 설정**  
   0달러부터 35달러까지 1달러 단위로 가격대를 나누었습니다.  
   예: `$0-$1`, `$1-$2`, ..., `$34-$35`

3. **Bin 레이블 부여 및 카운트**  
   `pd.cut()`을 활용하여 가격대별로 게임 수를 집계했습니다.

### 주요 해석

- **$0-$1 가격대에 게임이 가장 많음**: 무료 또는 매우 저렴한 게임이 Steam에 가장 많이 등록되어 있음.
- **$1~$5 사이에도 높은 밀집도**: 인디 게임, 할인 게임 등이 이 구간에 다수 포함됨.
- **$10 이상에서는 급격히 감소**: 고가 게임은 상대적으로 적은 수를 차지.

## 🧠 결론

Steam 플랫폼의 게임들은 대부분 저가 또는 무료 형태로 제공되며, 고가의 게임은 소수에 불과합니다. 이는 게임 유통 전략, 시장 수요, 인디 개발자의 접근성 등을 반영한 결과로 해석될 수 있습니다.
