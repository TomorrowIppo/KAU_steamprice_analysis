# 🎲 두 개의 정규분포 생성 및 시각화

이 프로젝트는 평균이 다른 두 개의 정규분포 데이터를 생성하여, 두 분포가 어떻게 겹치고 분리되는지 시각적으로 확인하는 데 목적이 있습니다.

## 📊 분석 목표

- 평균이 서로 다른 두 개의 정규분포 데이터를 생성합니다.  
- 두 분포의 분포 차이와 중첩 영역을 시각화하여, 데이터 분포 특성을 이해합니다.

## ⚙️ 사용된 도구

- Python 3.x
- numpy
- matplotlib
- seaborn

## 🧹 데이터 생성 과정

1. **시드 고정**  
   재현성을 위해 `np.random.seed(42)`로 난수 시드를 고정했습니다.

2. **정규분포 설정**  
   - Population 1: 평균(μ) = 50, 표준편차(σ) = 10  
   - Population 2: 평균(μ) = 55, 표준편차(σ) = 10  
   - 각 분포별 데이터 수는 1500개로 동일하게 설정했습니다.

3. **데이터 생성**  
   `np.random.normal()` 함수를 사용해 두 분포별 데이터를 각각 생성했습니다.

## 📈 시각화 결과

아래 그래프는 두 정규분포의 히스토그램과 커널 밀도 추정(kde)을 함께 나타냅니다.  
- 하늘색: Population 1 (μ=50)  
- 살구색: Population 2 (μ=55)  

이 그래프를 통해 두 분포의 평균 차이로 인해 분포가 서로 겹치면서도 구분되는 패턴을 확인할 수 있습니다.

## 🧠 해석 및 활용

- 평균이 다르더라도 표준편차가 같으면 두 집단이 일부 겹칠 수 있음을 시각적으로 확인할 수 있습니다.  
- 이와 같은 분포 시각화는 실제 데이터에서 그룹 간 차이를 탐색하거나, 가설 검정 시 기본 분포를 이해하는 데 활용됩니다.
