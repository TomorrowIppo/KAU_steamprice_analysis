# 📊 두 모집단 평균 차이의 신뢰구간 추정

이 프로젝트는 두 개의 정규 모집단에서 임의로 추출한 표본을 이용해 **모평균 차이(μ₁ − μ₂)**의 **95% 신뢰구간**을 다양한 가정(Z, t, Welch)에 따라 추정합니다.  
이를 통해 모집단 분산에 대한 가정이 신뢰구간 추정 결과에 어떤 영향을 주는지 비교 분석합니다.

## 🎯 분석 목표

- 두 정규 모집단에서 샘플을 추출하여 평균 차이를 계산합니다.
- 모집단 분산에 대한 세 가지 가정하에 95% 신뢰구간을 각각 추정합니다.
  - 모집단 분산을 아는 경우 (Z-distribution)
  - 분산은 모르지만 같다고 가정하는 경우 (t-distribution, pooled)
  - 분산도 모르고 같다고도 할 수 없는 경우 (Welch’s t-distribution)
- 각 추정법의 결과를 비교하여 차이를 해석합니다.

## 🗂️ 데이터 생성

- 두 모집단 모두 정규분포에서 생성
  - 모집단 1: 평균 50, 표준편차 10
  - 모집단 2: 평균 55, 표준편차 10
- 각 모집단 크기: 1,500
- 각 모집단에서 무작위로 샘플 추출
  - 표본 크기: 모집단 1 → 81개, 모집단 2 → 101개

## ⚙️ 사용된 도구

- Python 3.x  
- NumPy  
- SciPy  
- Matplotlib  
- Seaborn  

## 🔬 분석 과정

1. **모집단 시각화**  
   - 두 정규분포를 히스토그램 + KDE 곡선으로 비교

2. **표본 추출 및 통계량 계산**  
   - 샘플 평균, 샘플 표준편차, 표본 크기

3. **신뢰구간 계산**  
   - Z-distribution (모집단 분산을 아는 경우)  
   - t-distribution (분산은 같다고 가정)  
   - Welch’s t-distribution (분산이 다를 수 있음)

---

## 📈 결과 요약

- 샘플 평균 차이 (x̄₁ − x̄₂): **-4.23**

### 1. 모집단 분산을 아는 경우 (Z-distribution)
- 95% 신뢰구간: **(-7.15, -1.30)**

### 2. 분산이 같다고 가정하는 경우 (Pooled t-distribution)
- 95% 신뢰구간: **(-7.10, -1.35)**

### 3. 분산이 다를 수 있는 경우 (Welch’s t-distribution)
- 95% 신뢰구간: **(-7.08, -1.37)**

✅ **모든 방법에서 평균 차이는 0보다 작음**, 즉 **모집단 2의 평균이 유의미하게 더 큼**.

---

## 🧠 분석 해석

- 세 방법(Z, t, Welch)의 신뢰구간 결과는 **거의 유사**합니다.
- 이는 표본 크기(n₁=81, n₂=101)가 충분히 크고, 두 모집단이 모두 정규분포이기 때문입니다.
- Welch’s 방법은 **분산이 다를 가능성**까지 고려하므로 일반적으로 가장 보편적이며 안전한 선택입니다.
- 표본 평균 차이가 **-4.23**이고, 모든 신뢰구간이 **음수 구간에 있음** → 평균 차이는 통계적으로 유의미합니다.

---

## 📝 결론 요약

- **두 모집단 간 평균 차이는 유의미하게 존재**합니다.
- **분산에 대한 가정(Z, t, Welch)**은 표본 수가 충분할 경우 결과에 큰 차이를 주지 않지만, 작은 샘플에서는 중요한 요소가 될 수 있습니다.
- Welch 방법은 가장 일반화된 방법으로 추천됩니다.
