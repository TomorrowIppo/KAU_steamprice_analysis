## 🔢 문제 개요
- 두 모집단(Population 1, Population 2)으로부터 각각 표본 크기 81, 101개의 데이터를 무작위로 추출했습니다.
- 각 표본의 분산을 계산한 뒤, 모집단 분산 비율 σ₁² / σ₂²의 95% 신뢰구간을 추정하는 것이 목표입니다.

---

## 📊 샘플 정보
- 샘플 1
  - 표본 크기: n₁ = 81
  - 표본 분산: s₁² ≈ 88.812
- 샘플 2
  - 표본 크기: n₂ = 101
  - 표본 분산: s₂² ≈ 100.846
- 표본 분산 비율: 88.812 / 100.846 ≈ 0.881

---

## 📐 신뢰구간 계산 방법
- 자유도
  - df₁ = 81 − 1 = 80
  - df₂ = 101 − 1 = 100
- F-분포 분위수 (α = 0.05)
  - 하위 2.5% 분위수: F(0.025, 80, 100)
  - 상위 97.5% 분위수: F(0.975, 80, 100)
- 신뢰구간 공식: (s₁² / s₂²) / F_upper, (s₁² / s₂²) / F_lower
- 계산된 신뢰구간: (0.5824, 1.3449)

---

## ✅ 결과
- 95% 신뢰구간: (0.5824, 1.3449)

---

## 📌 해석
- 이 신뢰구간은 1을 포함하고 있으므로,
  - 두 모집단의 분산이 같다는 귀무가설을 기각할 수 없습니다.
- 즉, 두 모집단의 분산 차이가 통계적으로 유의하지 않을 가능성이 높습니다.
