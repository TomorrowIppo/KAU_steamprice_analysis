# KAU_steamprice_analysis

본 프로젝트는 한국항공대학교 소프트웨어학과 김선옥 교수님의 확률통계론 과목의 팀프로젝트 수행을 위해 기획되었습니다. 주어진 조건에 따라 통계적 추정을 실험하고, 신뢰구간 및 예측구간을 계산하여 통계적 해석을 수행합니다.

## 👥 팀 구성
- 김병진, 소프트웨어학과 2022125008
- 유승, 소프트웨어학과 2022125032

## 🛠 사용 도구
- 프로그래밍 언어는 Python으로 진행합니다.

## 📌 프로젝트 과업 요약

### 1. 데이터셋 수집
- 공개된 소스로부터 **1,000개 이상**의 데이터를 포함하는 데이터셋 다운로드
- 해당 데이터셋을 **모집단(population)** 으로 정의

### 2. 데이터셋 설명 및 시각화
- 데이터의 특성을 서술
- 적절한 **그래프 시각화** 수행 (히스토그램, boxplot, 분포 곡선 등)

### 3. 표본 추출 및 통계 분석
- 모집단에서 각각 **10개, 30개, 100개** 표본을 **무작위 추출**
- 각 표본에 대해:
  - 표본 평균 (𝑋̄), 표본 분산 (𝑆²) 계산
  - **모집단 평균에 대한 99% 신뢰구간** 계산
  - **모집단 분산에 대한 99% 신뢰구간** 계산
  - **미래 관측값에 대한 99% 예측구간** 계산 및 실제값과 비교

### 4. 두 정규분포 모집단 생성
- 서로 다른 평균(𝜇₁, 𝜇₂)과 **동일한 분산(𝜎₁² = 𝜎₂²)** 을 갖는 **정규분포 모집단** 2개 생성 (각각 1,000개 이상)
- **실제 데이터 vs 확률 분포 그래프 시각화**

### 5. 두 평균 간 차이에 대한 추정
- 모집단에서 각각 **81개, 101개** 표본 무작위 추출
- 표본 평균의 분포를 이용하여 **𝜇₁ - 𝜇₂의 95% 신뢰구간** 추정
  - 모집단 분산 **알려진 경우**
  - 모집단 분산 **모르지만 동일하다고 가정한 경우**
  - 모집단 분산 **모르는 경우**

### 6. 두 모집단 분산 비율 추정
- 앞에서 사용한 표본으로부터 **분산 비율 (𝜎₁² / 𝜎₂²)** 에 대한 **95% 신뢰구간** 추정
